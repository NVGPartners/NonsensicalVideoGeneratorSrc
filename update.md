> ## Milestone 7 (v1.0.3.0)
> Here are the changes in this update:
> - FFmpeg is now integrated into NVG for video playback, which makes previews much faster and smoother.
> - NVG is now open source on GitHub. You can view the source code [url=https://github.com/NVGPartners/NonsensicalVideoGeneratorSrc]here[/url].
> - Added a button to the top right corner of the overlay border to close NVG.
> - Added a button to the bottom right corner of the overlay border to make the current video fullscreen.
> - Added a video that plays when NVG starts, showing the KiwifruitDev logo.
> - Updated the default outro with the new KiwifruitDev logo.
> - Added support for Steam rich presence, which shows the current tab on your friends list.
> - Added an FPS unlock option to the debug menu (Ctrl+F3).
> - Added an option to open console.txt from the debug menu.
> - Updated credits and bundled more license files.
>   - Moved around certain Superstar users.
>   - Renamed eddsworldfan69420 to tehcartoonman22145.
>   - Added licenses for Nonsensical Video Generator (GPL-3.0 License), MonoGame.Extended (MIT License), MonoGame.Extended2 (BSD 3-Clause Clear License), and FFmpeg (GPL-3.0 License).
>   - Added attribution for Spitfire Audio.
> - Updated the localization files.
> Let me know if you have any issues with this update.
> Also, please help translate NVG into your language on [GitHub](https://github.com/KiwifruitDev/NonsensicalVideoGenerator/wiki/Localization)!
